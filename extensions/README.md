# solana-tools-lite-extensions

This crate contains protocol-specific implementations for [solana-tools-lite](https://crates.io/crates/solana-tools-lite).

## Features

- **Light Protocol (ZK Compression):** Analysis and privacy classification for ZK Compression transactions.

> **Note:** This is an internal helper crate for the `solana-tools-lite` CLI ecosystem. The extension interfaces are defined in the core `solana-tools-lite` crate.
